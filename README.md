# My Personal Website

This is the source code for my personal website. The project is built using **Next.js**, **React**, **Tailwind CSS**, and deployed for performance and scalability.

## Features
- Modern and responsive UI
- Fast loading and optimized SEO
- Metadata API integrated
- Sitemap generated and submitted

##  Tech Stack
- **Next.js**
- **React**
- **Tailwind CSS**
- **Vercel**
- **Metadata API**

## Project Structure
```
/app
  /components
  /api
  /styles
/public
```
